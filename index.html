Atipa is a high performance computing company that has been building Linux-based HPC clusters for well over a decade. They were seeking help in updating their website - which had not been updated in more than a few years. Firstly, a project scope was created that could clearly set expectations and to determine the timeline. Secondly, I provided better design techniques that helped drive more clients to the site - from the home page to the contact page and everywhere in between. The visual style that I used best reflected the brand and oranization as a whole.

American Disability Solutions is America's premier Social Security and Disability institution. They were seeking help in creating a static site that could convert page views to clients. Firstly, a project scope was put into place in order to implement a timeline. Secondly, a sitemap was created in order to better understand how the overall navigation should be structured. Lastly, was the development process. I built this responsive site from scratch - with the help from the front end framework - Bootstrap. I delivered a site that created a great lasting impression.

DragonBall Merchandise is a one stop shop for every DragonBall fan's needs - anywhere from action figures to video games. A sitemap was created in order to better understand how to organize the site - Homepage - Categories - Item - Contact.

I developed this site with Ruby on Rails as the backend framework and Bootstrap as the responsive HTML framework. The database is Postgres and is hosted on a cloud. I created everything, from the tables to the views. This site is a part of Amazon Associates LLC and has gained traction throughout the DragonBall community.
